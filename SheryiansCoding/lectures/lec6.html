EJS stands for Embedded JavaScript. It's a templating engine used in backend web development, especially with Node.js and Express.js.

when we use ejs

EJS is often used with the Express.js framework as a view engine

You can use EJS to create dynamic HTML email templates.

app.use(express.json());

app.use(express.urlencoded({ extended: true }));

app.set("view engine", "ejs");
